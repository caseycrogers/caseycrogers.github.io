Mesh Maker

Github Repository

Mesh Maker is a ~800 line Python add-in that I wrote for Autodesk Fusion360. It takes in a triangulated 3D object as an .stl file and converts it into a series of small interlocking and 3D printable triangles exported as a series of .stl files. Binary bit patterns embedded into each triangle guide the user through assembly.

Mesh Maker takes large complex objects with structures challenging to 3D printing and makes them easily printable on a consumer grade FDM machine. The result is both artistic and and funtional for rapid prototyping.

YourFault

Hackster Page

YourFault is an Android Wear and Mobile app that I designed for the "User Interfaces" course at UC Berkeley. The app was designed in Android Studio to match a project specification.

YourFault's mobile application uses the USGS API to push a notification screen to the Android Wear app when USGS detects an earthquake. The notification displays the earthquake location, magnitude and distance from the user. Tapping on the notification opens, on the mobile app, a Google maps view with a circle representing the mangitude and epicenter of the earthquake.

Wright J Whirlwind Radial Engine

Fusion 360 Gallery Page

I made this model of the Wright J Whirlwind Engine in collaboration with Daniele Grandi and Carlos Oyuela-Mora for the "Design For Autodesk" program. The model is based off of engineering drawings by Jan Elg.

As the team lead for the project, I:
Delegated work between the team members
Modeled and assembled all exterior casing of the engine (excluding rear gearbox casing)
Designed the interior parts of the cylinders, including the piston assembly, impeller and rocker valves
